🎨 CSS - CARD IMAGE

🔴 Edit Gambar Produk

/* SEBELUM */
.card-image {
  width: 100%;
  height: 200px;
  object-fit: cover;
  border-radius: 10px 10px 0 0;

/* SESUDAH - Gambar Lebih Tinggi */

💡 Tip:
  • object-fit: cover = Gambar dipotong sesuai ukuran container
  • object-fit: contain = Gambar full terlihat (ada space putih)

🔴 Excel: Product Grid Layout

Mengatur jumlah card per baris (responsive):

.products-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
  gap: 20px;
}

/* Artinya:
  - repeat(auto-fit, ...) = otomatis wrap ke baris berikutnya
  - minmax(280px, 1fr) = minimum width 280px, maksimal fill space
  - gap: 20px = jarak antar card
  
  Hasil: 
    Desktop (wide): 4 cards per baris
    Tablet: 2-3 cards per baris
    Mobile: 1 card per baris
*/

🎨 CSS untuk Setiap Bagian

🔴 BAGIAN 1: .product-card (Container Utama)

.product-card {
  background: white;              /* Latar belakang putih */
  border-radius: 12px;            /* Sudut bulat 12px */
  overflow: hidden;               /* Image tidak keluar dari sudut */
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);   /* Shadow subtle */
  transition: all 0.3s ease;      /* Smooth animation untuk hover */
}
💡 Tip Parameter:
  • border-radius: 12px = Ukuran sudut melengkung (12px bagus untuk card)
  • overflow: hidden = Potongan gambar yang keluar dari border-radius
  • box-shadow = Bayangan di bawah card (membuat depth)
  • transition = Animasi smooth saat hover

🔴 BAGIAN 2: .card-image (Gambar Produk)

.card-image {
  width: 100%;                    /* Full width card */
  height: 200px;                  /* Tinggi tetap 200px */
  object-fit: cover;              /* Gambar dipotong, tidak stretch */
  display: block;                 /* Remove spacing (inline element) */
}
⚠️ Important:
  • Jangan gunakan object-fit: stretch - gambar akan jelek!
  • Gunakan object-fit: cover untuk product cards
  • Height harus fixed (bukan auto) supaya semua card sama ukuran

🔴 BAGIAN 3: .card-body (Wrapper Text)

.card-body {
  padding: 20px;                  /* Jarak dalam 20px */
  background: white;
  display: flex;                  /* Flexbox untuk layout */
  flex-direction: column;         /* Stack vertikal */
}

🔴 BAGIAN 7: .btn-order (Tombol Pesan)

Button harus clickable, eye-catching, dan user-friendly!

DEFAULT
.btn-order {
  background: #c7319c;     /* Pink */
  color: white;
  border: none;
  padding: 12px 24px;
  border-radius: 8px;
  font-size: 1em;
  font-weight: 600;
  cursor: pointer;
}
+ HOVER EFFECT
.btn-order:hover {
  background: #a71f6a;         /* Lebih gelap */
  transform: translateY(-2px); /* Naik sedikit */
  box-shadow: 0 4px 12px 
    rgba(199,49,156,0.3);      /* Shadow pink */
}
1 Padding: 12px 24px = compact button fit di card
2 Border-radius: 8px = slightly rounded (professional look)
3 Cursor: pointer = tangan icon (user tahu bisa diklik!)
4 :hover = state saat mouse di button (PENTING!)

🎨 BONUS: Card Hover Animation

/* Saat mouse hover ke card, card naik & shadow tebal */
.product-card:hover {
  transform: translateY(-8px);    /* Naik 8px */
  box-shadow: 0 12px 24px rgba(0,0,0,0.15);  /* Shadow besar */
}
💡 Kombinasi Hover:
  • Card naik (translateY)
  • Shadow lebih tebal (box-shadow)
  • Tombol berubah warna (:hover)
  • = Semua smooth dengan transition!
