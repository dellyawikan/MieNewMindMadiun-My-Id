💡 Tips untuk Pemula: Klik salah satu halaman di bawah, kemudian baca dari atas ke bawah untuk melihat semua elemen yang bisa diedit. Setiap bagian dijelaskan dengan kode contoh!

Pilih Halaman yang Ingin Anda Edit:

📄 Halaman Aktif: Home (index.html) - Halaman pertama yang dilihat pengunjung website

🏠 HALAMAN HOME (index.html)

Halaman utama website yang pertama kali dilihat pengunjung. Berisi hero section, produk pilihan, promo, dan CTA penting.

1️⃣ NAVBAR (Bagian Atas)

File: index.html (Baris 1-30)

Struktur Navbar:

  • Logo: Gambar di sebelah kiri (assets/logo/logo.png)
  • Menu Links: Beranda, Menu, Tentang Kami, Lokasi, Reservasi, Karir, Franchise
  • Button: WhatsApp atau Login

📝 Cara Edit Logo:

<img src="assets/logo/logo.png" class="brand-logo">

Ganti path menjadi logo baru Anda.

📝 Cara Edit Menu Links:

<nav class="main-nav">
  <a href="index.html">Beranda</a>
  <a href="menu.html">Menu Kami</a>
  <a href="about.html">Tentang Kami</a>
</nav>

Edit text di dalam tag <a> untuk ubah nama menu.

2️⃣ HERO SECTION (Bagian Besar di Atas)

File: index.html (Baris 31-70)

Bagian-Bagian Hero:

  • Background Image: Gambar besar di belakang
  • Judul (Title): Text besar "Selamat Datang"
  • Subtitle: Text kecil di bawah judul
  • CTA Button: Tombol "Lihat Menu"

📝 Cara Ubah Background Image:

<section class="hero" style="background-image: url('assets/pages/hero-home.jpg');">

Ganti path 'assets/pages/hero-home.jpg' dengan gambar baru.

📝 Cara Ubah Judul Hero:

<h1 class="hero-title">Selamat Datang di Mie Newmind</h1>

Edit text di dalam <h1>...</h1>

📝 Cara Ubah Subtitle:

<p class="hero-subtitle">Mie Pedas Terbaik di Kota</p>

Edit text di dalam <p>...</p>

📝 Cara Ubah Button:

<a href="#menu-section" class="btn btn-primary">Lihat Menu Kami</a>

Edit text untuk mengubah label tombol.

3️⃣ PROMO/FEATURED SECTION

File: index.html (Baris 71-150)

Section ini menampilkan menu pilihan atau promo spesial dalam bentuk card.

📝 Struktur Card:

<div class="product-card">
  <img src="assets/products/mie-special.jpg" alt="Mie Special">
  <h3>Mie Spesial</h3>
  <p>Deskripsi singkat produk</p>
  <div class="price">Rp15.000</div>
  <button>Lihat Detail</button>
</div>

📝 Cara Ubah Card:

  • Gambar: Edit path di src="..."
  • Nama: Edit text di <h3>
  • Harga: Edit di <div class="price">
  • Deskripsi: Edit di <p>

4️⃣ FOOTER (Bagian Bawah)

File: index.html (Baris terakhir)

Bagian Footer:

  • Informasi Toko: Deskripsi singkat
  • Kontak: Nomor WhatsApp, Email
  • Alamat: Lokasi toko
  • Links: Link ke halaman lain
  • Social Media: Link Instagram, Facebook, dll

📝 Cara Edit Kontak WhatsApp:

<a href="https://wa.me/6285604030473">Hubungi Kami</a>

Ganti nomor 6285604030473 dengan nomor WhatsApp Anda.

📝 Cara Edit Alamat:

<p>📍 Jalan Merdeka No. 123, Kota Anda</p>

Edit alamat sesuai lokasi toko.

💡 Tips: Halaman home adalah yang paling penting karena pertama kali dilihat pengunjung. Pastikan hero section menarik, promo jelas, dan kontak mudah ditemukan!
