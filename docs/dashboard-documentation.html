Fitur-Fitur Dashboard

1. Dashboard (Home)

Status Keseluruhan Website

Di halaman utama dashboard, kamu bisa lihat status semua aspek website Mie Newmind dalam satu pandangan. Ada info tentang domain status (aktif/tidak), uptime (99.95%), SSL certificate status, dan security grade (B+).

2. Domains

Kelola Domain Kamu

Tab ini buat manage domain yang udah terdaftar. Kamu bisa lihat kapan domain kamu bakal expired, siapa registrarnya, dan bisa buat renewal atau update WHOIS info. Semua domain yang terdaftar bakal muncul di sini dengan detail lengkap.

3. DNS Management

Atur DNS Records

Ini tuh fitur yang super penting! Di sini kamu bisa manage DNS records yang ngatur mana server yang bakal handle domain kamu. Ada beberapa tipe record seperti A, CNAME, TXT, MX, dan NS. Jangan sembarangan edit karena bisa bikin website error!

DNS Records yang Ada:

  • A Record: 172.66.46.223 (pointing ke Cloudflare)
  • CNAME: www record pointing ke Cloudflare Pages
  • TXT: For verification purposes
  • MX: Buat email handling
  • NS: Nameserver dari Cloudflare

4. Hosting

Info Provider Hosting

Di tab ini kamu bisa lihat info lengkap tentang provider hosting kita yang adalah Cloudflare Pages. Ada info tentang uptime, response time, storage, dan fitur-fitur yang tersedia kayak HTTP/2, Brotli compression, dan edge caching yang bikin website super cepat!

5. SSL/TLS

Security Certificate Management

SSL/TLS ini yang bikin website kita aman dengan enkripsi. Di sini kamu bisa lihat status certificate kita (yang sekarang udah VALID), siapa issuernya (Let's Encrypt), tipe certificate (Wildcard), dan kapan bakal expired. Ada juga TLS configuration yang bisa kamu atur.

6. Analytics

Traffic dan Performance Metrics

Pengen tahu berapa banyak orang yang akses website? Di sini tempatnya! Analytics menunjukin jumlah requests, bandwidth yang dipakai, response time, dan cache ratio. Ada juga tabel yang nunjukin halaman mana yang paling banyak dikunjungi.

7. Billing

Informasi Pembayaran

Tab billing menunjukin informasi tentang plan yang kita pake. Kita pake Cloudflare Pages yang gratis (Rp 0/bulan), jadi unlimited semuanya! Ada juga riwayat invoice kalau kita pernah bayar untuk upgrade.

8. Settings

Pengaturan Akun

Di settings kamu bisa manage account information, security settings, dan notification preferences. Ada juga information tentang 2FA status, API keys, dan login activity. Jangan lupa buat enable 2FA buat keamanan extra!

Cara Logout

Kalau udah selesai menggunakan dashboard dan pengen logout, ada beberapa cara:

  1. Scroll ke bawah di sidebar
  2. Klik tombol LOGOUT yang berwarna merah
  3. Atau tekan keyboard shortcut Ctrl + L (Windows) atau Cmd + L (Mac)
  4. Konfirmasi logout di popup dialog
  5. Kamu bakal di-redirect ke halaman login

Tips dan Trik

Pro Tips:

  • Jangan edit DNS records sembarangan kecuali kamu tahu apa yang kamu lakukan
  • Enable 2FA di settings untuk keamanan maksimal
  • Check analytics regularly buat tahu traffic website
  • Monitor SSL certificate expiry date biar nggak tiba-tiba expired
  • Gunakan keyboard shortcut Ctrl+L buat logout yang lebih cepat

Troubleshooting

Lupa Password?

Password adalah: newmind2024 (restaurant name + year). Kalau masih lupa, hubungi admin yang setdapat akses ke source code.

Website Down?

Check di tab Hosting atau Analytics apakah ada masalah dengan provider. Kalo nggak ada masalah di sana, kemungkinan masalah di kode website. Check console di browser (F12) buat error details.

DNS Tidak Berubah?

DNS changes bisa butuh waktu sampai 24-48 jam untuk propagate ke seluruh internet. Jadi sabar dikit ya, nggak perlu panic!

Fitur Keamanan

Security Headers yang Terinstall:

  • Content Security Policy (CSP): Proteksi dari XSS attacks
  • Strict Transport Security: Force HTTPS connection
  • X-Frame-Options: Prevent clickjacking
  • X-Content-Type-Options: Prevent MIME type sniffing
